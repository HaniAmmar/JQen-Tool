# JQen Tool

**JQen Tool** is a streamlined utility designed for **testing templates** authored with the expressive syntax of **[Qentem Engine](https://github.com/HaniAmmar/Qentem-Engine)**.

This tool facilitates rapid validation and iteration of template logic, making it an essential asset for developers.
